# TaskReminder

Aplicación web para la gestión de tareas académicas.

## Descripción
TaskReminder permite a los usuarios crear, visualizar y gestionar tareas de manera sencilla. El proyecto se desarrolla por sprints, iniciando con un módulo funcional en Node.js y MySQL.

## Estado del proyecto
- Sprint 3: Módulo funcional de gestión de tareas (Node.js + MySQL)
- Próximo: Migración a Laravel

## Tecnologías
- Node.js
- Express
- MySQL
- (Futuro) Laravel

## Funcionalidades actuales
- Crear tareas
- Listar tareas
- Marcar como completadas
- Eliminar tareas
